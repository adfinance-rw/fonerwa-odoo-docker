# Contract Management System

A comprehensive contract management system for Odoo 18.

## Features

- Contract creation and management
- Contract templates
- Contract tracking and monitoring
- Contract renewal management
- Status tracking (Draft, Active, Expired, Cancelled)
- Customer management integration
- Contract value tracking
- Notes and descriptions

## Installation

1. Copy the module to your Odoo addons directory
2. Update the apps list
3. Install the module from the Apps menu

## Usage

1. Navigate to Contract Management > Contracts
2. Create new contracts
3. Track contract status and expiration dates
4. Manage contract renewals

## Technical Details

- Compatible with Odoo 18.0
- Uses mail.thread for activity tracking
- Includes proper security access controls
- Follows Odoo development best practices
